Doing Computational Social Science

John McLevey
Sage, London, 2022

A practical guide to computational methods for social scientists. This book covers the full research workflow from data collection and processing to analysis and visualization, with an emphasis on reproducibility and principled data processing.

Topics Covered

Data Collection

Web scraping, APIs, and working with digital behavioral data

Text Analysis

From foundations to transformer-based approaches

Network Analysis

Methods for analyzing social and information networks

Simulation

Agent-based modeling and generative approaches

Companion Package

The book is accompanied by the dcss Python package, which includes curated datasets and utility functions for data processing, statistical modeling, and machine learning.
