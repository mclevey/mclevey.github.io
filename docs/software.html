Software

Open-source Python packages for computational social science research.

Packages

dcss

GNU GPL 2 · John McLevey, Tyler Crick, Pierson Browne

Python package accompanying Doing Computational Social Science. Includes curated datasets and utility functions for data processing, statistical modeling, and machine learning.

metaknowledge (mk)

GNU GPL 2 · Reid McIlroy-Young and John McLevey

Python package for quantitative and network research in the sociology of science and information science. First released 2015, under active development.

pdpp

MIT License · Pierson Browne, Tyler Crick, Rachel Wood, John McLevey

Principled data processing with Python. Tools for reproducible data workflows. First released 2015, under active development.

Nate

MIT License · John McLevey, Pierson Browne, Tyler Crick

Network analysis with text. Python package for network-based text analysis.

simkit

MIT License · John McLevey

Probabilistic and agent-based computational models of opinion dynamics, diffusion, learning, and influence. Currently in beta.

tidyextractors

MIT License · Joel Becker

Extract quantitative datasets from source code repositories, email mbox files, and social media APIs.

recordlinkage

Contributions to Jonathan de Bruin's Python package for record linkage. NetLab contributions focused on comparison and fusion algorithms for cross-sectoral collaboration network research.
